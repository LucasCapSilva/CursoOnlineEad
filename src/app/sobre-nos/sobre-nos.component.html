Sobre o Projeto

Plataforma educacional desenvolvida como Single Page Application (SPA) para gerenciamento completo de cursos online, engajamento de alunos e acompanhamento de trilhas de aprendizado.

Propósito

O objetivo principal desta aplicação é centralizar a experiência de aprendizado, oferecendo um ambiente integrado onde instrutores e alunos podem interagir, gerenciar entregas, acessar conteúdos multimídia e monitorar o desempenho acadêmico e de bem-estar. O sistema foi projetado para suportar trilhas de ensino de ponta a ponta.

Funcionalidades Principais

  • Gestão de usuários e perfis com controle de acesso (RBAC).
  • Criação, edição e acompanhamento de atividades e exercícios.
  • Fórum de discussões integrado para suporte e interação.
  • Controle de presenças e integração com salas virtuais (Zoom).
  • Monitoramento contínuo do bem-estar dos participantes.
  • Ferramenta de geração e visualização de currículos profissionais.

Arquitetura e Tecnologias

Arquitetura de Software

A aplicação segue uma arquitetura baseada em componentes utilizando o framework Angular. A estrutura é altamente modular, dividida em features específicas (módulos de alunos, instrutores, administradores). A comunicação com o back-end é realizada de forma assíncrona através de serviços injetáveis que consomem APIs RESTful, garantindo escalabilidade e separação de responsabilidades. O estado da aplicação é gerenciado com técnicas reativas (RxJS).

Stack Tecnológico
Angular TypeScript Bootstrap 4 HTML5 CSS3 RxJS

Fluxo de Navegação e Uso

O acesso à plataforma é protegido por autenticação via token JWT. Após o login, o usuário é redirecionado para a Home/Dashboard baseada em seu nível de acesso:

  1. Acesso a Conteúdos: Navegação pelas trilhas de módulos, onde vídeos, PDFs e materiais de leitura estão organizados sequencialmente.
  2. Atividades: Acesso à área de exercícios para submissão de respostas e visualização de feedback dos instrutores.
  3. Interação: Utilização do Fórum para tirar dúvidas e da área de Links Úteis para acessar salas ao vivo.
  4. Perfil: Atualização de informações cadastrais e construção do currículo profissional para exportação.
